Talk Testing & Automation Demo. [Live Demo](http://kios-demo.now.sh)


## Development Setup

### Install Project Dependencies
```sh
npm install
```

### Install Apps/Packages Dependencies
```
npx lerna bootstrap
```

### Run The Apps
```sh
npm run dev
```

## Test

### Running Unit & Integration Tests
```sh
npm run test
```

### Running E2E Tests
```sh
npm run test:e2e
```
